HTML-текст


HTML- текст представлений в специфікації елементами для форматування і угруповання тексту. Дані елементи є контейнерами для тексту і не мають візуального відображення.

Елементи для форматування тексту несуть смислове навантаження і зазвичай задають для тексту, укладеного всередину, стильове оформлення, наприклад, виділяють текст жирним шрифтом або відображають його шрифтом іншого сімейства (властивість font-family).

Грамотно відформатований текст дає зрозуміти пошуковим системам, які слова несуть важливе смислове навантаження, по яким з них переважно ранжувати веб-сторінку в пошуковій видачі. Вся текстова інформація, яка відображається на нашому каталозі, зв'яжіться всередині елемента body.


1. HTML-елементи для заголовків


Заголовки є важливими елементами веб-сторінки, вони впорядковують текст, формуючи його візуальну структуру. елементи <h1>...<h6> повинні використовуватися тільки для виділення заголовків нового розділу або підрозділу.

При використанні заголовків необхідно враховувати їх ієрархію, тобто за <h1> повинен слідувати <h2> і т.д. Також не рекомендується вкладати в заголовки інші елементи.




1.1. Елемент <h1>

Заголовок самого верхнего уровня, на странице рекомендуется использовать только один раз, по частично дублируя заглавие страницы. Елемент <h1> должен быть уникальным для каждой страницы сайта.

Рекомендуется прописывать в начале статьи, используя ключевое слово в тексте заголовка. Размер шрифта в браузере равен 2em, верхний и нижний отступ по умолчанию 0.67em.

1.2. Елемент <h2>

Являє підзаголовки елемента <h1>. Розмір шрифту в браузері дорівнює 1.5em, верхній і нижній відступ за замовчуванням 0.83em.

1.3. Елемент <h3>

Показує підзаголовки елемента <h2>. Розмір шрифту в браузері дорівнює 1.17em, верхній і нижній відступ за замовчуванням 1em.

1.4. Елемент <h4>, <h5>, <h6>

Позначають підзаголовки четвертого, п'ятого і шостого рівня. Розмір шрифту в браузері дорівнює 1em / 0.83em / 0.67em, верхній і нижній відступ за замовчуванням 1.33em / 1.67em / 2.33em відповідно.



2. Елементи для форматування тексту



2.1. Елемент <b>

Задає полужирное накреслення шрифту. Виділяє текст без акценту на його важливість.

2.2. Елемент <em>

Відображає шрифт курсивом, надаючи тексту значимість.

2.3. Елемент <i>

Відображає шрифт курсивом.

2.4. Елемент <small>

Зменшує розмір шрифту на одиницю по відношенню до тексту.

2.5. Елемент <strong>

Задає напівжирний шрифт шрифту, відноситься до тегам логічної розмітки, вказуючи браузеру на важливість тексту.

2.6. Елемент <sub>

Використовується для створення нижніх індексів. Зрушує текст нижче рівня рядки, зменшуючи його розмір.

2.7. Елемент <sup>

Використовується для створення ступенів. Зрушує текст вище рівня рядки, зменшуючи його розмір.

2.8. Елемент <ins>

Виділяє текст в новій версії документа, підкреслюючи його.

Для элемента доступны атрибуты cite и datetime.

2.9. Елемент <del>

Перекреслює текст. Використовується для виділення тексту, видаленого з документа.

Для элемента доступны атрибуты cite и datetime.

2.10. Елемент <mark>

Застосовується для виділення фрагментів тексту в довідкових цілях, фарбуючи блок символів жовтим кольором.



3. Елементи для введення «компьютерного» тексту



3.1. Елемент <code>

Служить для виділення фрагментів програмного коду. Відображає текст моноширінним шрифтом.

3.2. Елемент <kbd>

Зазначає фрагмент як вводиться користувачем з клавіатури. Відображає текст моноширінним шрифтом.

3.3. Елемент <samp>

Застосовується для виділення результату, отриманого в ході виконання програми. Відображає текст моноширінним шрифтом.

3.4. Елемент <var>

Виділяє імена змінних, відображаючи текст курсивом.

3.5. Елемент <pre>

Дозволяє вивести текст на екран, зберігши початкове форматування. Прогалини і переноси рядків при цьому не видаляються.



4. Елементи для оформления цитат и определений



4.1. Елемент <abbr>

Застосовується для форматування абревіатур. Браузером зазвичай підкреслюється пунктирною лінією. Розшифровка скорочення здійснюється за допомогою атрибута title, вона з'являється при наведенні курсору миші на текст.

4.2. Елемент <bdo>

Використовується для зміни поточного напрямку тексту.

Для элемента доступен атрибут dir.

4.3. Елемент <blockquote>

Виділяє цитати всередині документа, виділяючи його відступами і переносами рядків.

Для элемента доступен атрибут cite.

4.4. Елемент <q>

Використовується для виділення коротких цитат. Браузерами полягає в лапки.

Для элемента доступен атрибут cite.

4.5. Елемент <cite>

Застосовується для виділення цитат, назв творів, виносок на інші документи.

4.6. Елемент <dfn>

Дозволяє виділити текст як визначення. Незважаючи на наявність даного елемента, рекомендується виділяти текст силами CSS.

Для элемента доступен атрибут title.



5. Абзаци, перенос тексту



5.1. Елемент <p>

Розбиває текст на окремі абзаци, відокремлюючи один від одного порожнім рядком. Браузер автоматично додає верхнє і нижнє зовнішнє поле margin, рівне 1em,при цьому поля сусідніх абзаців «схлопываются».

5.2. Елемент <br>

Переносить текст на наступний рядок, створюючи розрив рядка.

5.3. Елемент <hr>

Використовується для поділу контента на веб-сторінці. Відображається у вигляді горизонтальної лінії.
