Приємного дозвілля

10 гарячих клавіш VS Code, які прискорять вашу роботу

By Dima Pakin & Vika Jolysh Life hacks

Швидкий доступ - добро, користь і економія часу. Давайте розберемося, як з їх допомогою трохи спростити собі життя в Visual Studio Code.


Швидко додати коментар


Windows — Ctrl + /

Зазвичай щоб додати коментар, потрібно перейти в початок рядка і дописати туди два слеша. Це повільно. Швидше натиснути в будь-якому місці рядка Ctrl + /, і рядок закоментуйте, якщо вона ще не. А якщо вже так, то коментар зникне.




Перейти до рядка під номером


Windows — Ctrl + G

Іноді отладчики раптово повідомляють, що на рядку 953 трапилася непоправна інфернальна помилка всесвіту. Щоб не вишукувати вручну, не гортати і не витрачати на це час, натисніть Ctrl + G, введіть номер рядка і спокійно рятуйте всесвіт.




Поміняти рядок місцями з сусідніми


Windows — Alt + ↑ / ↓

Трапляється таке, що ваш алгоритм - НЕ алгоритм. Все пішло не так, а замість числа 42 вивелося повідомлення «Дані видалені». Кому-то для цього потрібно переписати весь код, але іноді буває досить поміняти рядки місцями і все запрацює.

Alt і стрілки змінюють поточний рядок місцями з сусідніми..




Дублювати рядок


Windows — Shift + Alt + ↓ / ↑

Це для тих, хто не хоче копіювати і вставляти кілька разів одне і те ж.




Відформатувати документ


Windows — Shift + Alt + F

VS Code запропонувати встановити розширення або вибрати з існуючих, а потім красиво отформатирует документ - розставить по місцях блоки, дужки, і зробить рядки коду читаються.
