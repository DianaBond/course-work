HTML-атрибути

HTML-атрибути повідомляють браузеру, яким чином повинен відображатися той чи інший елемент сторінки. Атрибути дозволяють зробити більш різноманітними зовнішній вигляд інформації, що додається за допомогою однакових тегів.

Значення атрибута полягає в лапки "". Назви і значення атрибутів не чутливі до регістру, але, тим не менш, рекомендується набирати їх в нижньому регістрі.


Глобальні атрибути


Глобальні атрибути, наведені в таблиці нижче, можуть бути використані для будь-якого HTML-елемента, хоча деякі з них можуть не надавати на елементи ніякого впливу.


Атрибут
Опис, прийняте значення
accesskey Генерує поєднання клавіш для доступу до поточного елементу. Складається з розділеного пробілами списку символів. Браузер в першу чергу вибирає ті клавіші, які існують на розкладці клавіатури. Застосовується до наступних елементів: <a>, <area>, <button>, <input>, <label>, <legend>, <textarea>.
Прийняті значення: перелік назв клавіш.
class Визначає ім'я класу для елемента (використовується для визначення класу в таблиці стилів).
Прийняті значення: імя класа.
contenteditable Визначає, чи може користувач редагувати вміст (контент). Дозволяє перетворити будь-яке поле HTML в редагований елемент.
Прийняті значення: true/false.
dir Визначає напрямок тексту контенту в елементах <bdo> и <bdi>.
Прийняті значення: ltr/rtl/auto.
draggable Визначає, чи може користувач перетягнути елемент.
Прийняті значення: true/false/auto.
hidden Вказує на те, що елемент повинен бути прихований.
Прийняті значення: hidden.
id Визначає унікальний ідентифікатор елемента.
Прийняті значення: id — идентификатор элемента.
lang Визначає код мови вмісту (контенту) в елементі.
Прийняті значення: код язика.
spellcheck Вказує, чи підлягає вміст елемента перевірці орфографії і граматики.
Прийняті значення: true/false.
style Упоказувала на код CSS, яка застосовується для оформлення елемента.
Прийняті значення: код CSS.
tabindex Визначає порядок переходу до елементу за допомогою клавіші TAB.
Прийняті значення: порядковий номер.
title Визначає додаткову інформацію про елемент, задаючи підказку для сторінки.
Прийняті значення: текст.
translate Дозволяє або забороняє переклад тексту всередині елемента.
Прийняті значення:yes/no.
